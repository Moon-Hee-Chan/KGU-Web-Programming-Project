# 자취생들을 위한 식재료 관리 및 레시피 웹 플랫폼
1인 가구, 특히 자취생에게 도움을 줄 수 있는 가성비 냉장고 가이드

## 사용한 기술
- **HTML5**
- **CSS3**
- **Tailwind CSS** (CDN)
- **JavaScript**
- **jQuery** (CDN)
- **Lucide Icons** (CDN)

## 주요 기능
- **레시피 찾기 :** 가성비 음식의 상세 조리법 및 예상 비용, 카테고리별 필터링 기능 제공
- **소분 방법 찾기 :** 대용량 식재료, 또는 자주 남는 식재료의 올바른 소분·보관법 제공
- **나의 냉장고 :** 현재 보유 중인 식재료의 종류·개수 파악 가능 (사용자가 추가 혹은 삭제할 수 있음.)

## 실행 방법
본 프로젝트는 별도의 서버 구축이나 빌드 과정 없이, 웹 브라우저에서 즉시 실행 가능

1. 터미널 또는 Git Bash를 열고 아래 명령어를 입력하여 프로젝트를 다운로드
2. 프로젝트 폴더 내의 '웹프로그래밍_프로젝트_최종본.html' 파일을 원하는 웹 브라우저로 더블 클릭하여 실행
